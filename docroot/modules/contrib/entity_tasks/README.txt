CONFIGURATION
-------------

 * Configure user permissions in Administration » People » Permissions:

    - Administer the entity tasks view mode
      Users in roles with the "Administer the entity tasks view mode"
      permission will be able to adjust the view mode of the toolbar
      integration.

    - Use entity tasks
      Users in roles with the "Use entity tasks" permission will be able
      to use the entity tasks.

 * Customize the display mode in Administration » Configuration » Entity
   Tasks Configuration

 * Add the Entity tasks block in Administration » Structure » Block layout
